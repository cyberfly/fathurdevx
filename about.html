About FathurDevX

FathurDevX is a modern web development template built with the latest technologies to provide a solid foundation for your projects.

Technology Stack

  • • Vite - Lightning-fast build tool and dev server
  • • Tailwind CSS - Utility-first CSS framework
  • • Alpine.js - Lightweight JavaScript framework
  • • vite-plugin-html - Multi-page support with shared layouts

Features

  • ✓ Multi-page architecture with shared base layout
  • ✓ Fast development with hot module replacement
  • ✓ Optimized production builds
  • ✓ GitHub Pages deployment ready
